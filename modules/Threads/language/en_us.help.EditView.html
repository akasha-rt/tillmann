Threads

Create one or more threads of discussion on a specific opportunity, bug, or case for a forum.

  • Title. Enter a title for the thread.
  • Body. Enter the contents of the thread.
  • Sticky. Select this box to create a sticky thread, which is always be at the top of the Thread list. When there are multiple stickies, the system lists the latest one at the top of the Thread list, followed by the others.

Click Save to create the thread.
