Foros

Utilice el módulo de foros para crear foros como medio de colaboración en general, técnica, o temas de ventas entre los usuarios dentro de su organización. Para cada foro, puede crear uno o más hilos de discusión sobre una oportunidad específica, error, o el caso.

La página de los foros Inicio muestra la siguiente información::

  • Una búsqueda sub-panel donde se puede entrar en un foro o el hilo de título y nombre de usuario para buscar un tema específico.
  • Una sección de accesos directos que muestra las siguientes opciones:
  • Foros. Haga clic en esta opción para ver una lista de foros existentes.
  • Crear foro. clic en esta opción para crear un nuevo foro.
